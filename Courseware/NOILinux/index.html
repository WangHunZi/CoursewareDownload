NOILinux-slim

虚拟机镜像 (VirtualBox) 在此下载。安装 VirtualBox 后，双击 ova 文件即可导入。

Features

  • 低配置机器 (包括优盘) 也可流畅运行。
  • 压缩镜像 2GB、解压 5 GB，移除非必要功能。
  • 更新 Linux 内核到 5.15，较新的机器也有驱动。
  • 仅保留 Visual Studio Code、Code::Blocks、Emacs、Vim 等主流开源编辑器/IDE。
  • user 密码：user，sudo 免密码。
  • 适合机房批量安装，安装时建议还原或去除 sudo 权限。
